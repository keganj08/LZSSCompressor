Lempel-Ziv Text Compressor

by Kegan Johnson

The LZSS compression algorithm losslessly reduces file sizes by decreasing redundancy. It scans for repeated data, then attempts to replace the repeats with a shorter token that refers back to the first instance of that data. These tokens are called length-distance pairs, as they indicate the length of the repeated data, and the distance one must go back to retrieve the original instance of it. Here, you can try out my implementation of LZSS for text.
