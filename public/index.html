About this tool

The LZSS compression algorithm losslessly reduces file sizes by decreasing redundancy. It scans for repeated data, then attempts to replace the repeats with a shorter token that refers back to the first instance of that data. These tokens -- represented here in the form {d,l} -- are called length-distance pairs, as they indicate the length of the repeated data, and the distance one must go back to retrieve the original instance of it.

Here you can use my implemenation of LZSS to reduce your text file sizes. Simply upload a text file or enter some text yourself, then click Encode or Decode. You can also download the results to store your file in the encoded format.
